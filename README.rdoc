== Wat is Lifefire?

-et lifefire kunnen -ensen de gezondheid verbeteren door een overzicht te krijgen van dagelijkse (on)gezonde activiteiten. Op basis van deze activiteiten worden er ook adviezen en kar-a gegeven.
